Pegá la URL de tu Google Apps Script para sincronizar y ver los mismos datos en todos tus dispositivos.

Ver instrucciones paso a paso →

Conectar Google Sheets
1. Abrí sheets.new y creá una hoja nueva.
2. Andá a Extensiones → Apps Script
3. Borrá el código existente y pegá el contenido del archivo google-apps-script.js
4. Guardá (Ctrl+S) y hacé clic en Implementar → Nueva implementación
5. Tipo: Aplicación web — Acceso: Cualquier persona
6. Copiá la URL y pegala en Configuración de esta app.
